# 🌍 Travel Guide AI

**Travel Guide AI** is an AI-powered travel assistant built using **Python**, **Streamlit**, and the **Groq API**. The chatbot helps users plan their trips by providing destination recommendations, travel budgets, hotel suggestions, food recommendations, travel itineraries, and useful travel tips.

---

## ✨ Features

* 🌍 Tourist Place Suggestions
* 💰 Budget Estimation
* 🏨 Hotel Recommendations
* 🍽️ Food Recommendations
* 🚗 Transport Options
* ☀️ Best Time to Visit
* 🧳 Packing Checklist
* 📅 Personalized Travel Itinerary
* 💡 Safety & Travel Tips
* 🤖 AI-Powered Travel Assistance

## 🚀 Installation

### 1. Clone the Repository

```bash
git clone https://github.com/your-username/TravelGuideAI.git
```

### 2. Open the Project Folder

```bash
cd TravelGuideAI
```

### 3. Install Required Libraries

```bash
pip install -r requirements.txt
```

### 4. Add Your Groq API Key

Create a `.env` file and add:

```env
GROQ_API_KEY=your_groq_api_key_here
```

### 5. Run the Project

```bash
streamlit run app.py
```
